# NYC Airbnb Market Analysis and Visualization

## Overview
This project analyzes and visualizes the NYC Airbnb market using Excel. The dataset contains information about Airbnb listings, pricing, availability, and customer preferences.

## Features
- Data cleaning and preprocessing in Excel
- Exploratory data analysis (EDA) using Excel functions and pivot tables
- Data visualization using Excel charts and graphs
- Insights and trends identification for the Airbnb market

## Technologies Used
- Microsoft Excel

## Dataset
The dataset used in this project includes Airbnb data for NYC, covering various attributes such as listing prices, neighborhood trends, availability, and customer reviews. 


## Usage
1. Open the Excel file: `airbnb_analysis.xlsx`.
2. Navigate through different sheets for data cleaning, analysis, and visualizations.
3. Explore insights using pivot tables and interactive charts.

## Visualizations
The project includes various visualizations such as:
- Bar charts showing pricing distribution
- Line graphs analyzing occupancy trends
- Pie charts illustrating neighborhood market share

## Insights on NYC Airbnb Market
To get underlying information and insights on the NYC Airbnb market and customer preferences.

This will help us to:
- Improve the company's business model.
- Help Airbnb make data-driven decisions about pricing, marketing, and customer experience.
- Create targeted strategies to increase company revenue.
